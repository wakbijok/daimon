B: Clusters with Sub-items

Each cluster expands in sidebar to show Nodes/VMs/Containers/Storage as sub-items. Like current Phase 4 "Cluster" section but per-cluster. Deeper sidebar, less horizontal tabs.

Option A (Recommended)

  • Sidebar stays clean — just cluster names
  • Horizontal tabs match PegaProx / PVE pattern
  • Scales well with many clusters
  • Main pane has more room for data

Option B

  • See all sections at a glance — no need to click first
  • Familiar from Phase 4 skeleton
  • Gets long with 3+ clusters (sidebar overflow)
  • Duplicates current DataGate pattern
